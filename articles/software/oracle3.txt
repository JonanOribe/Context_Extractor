¿Qué es Autonomous Database?

Oracle redefine la gestión de datos con la primera base de datos autónoma del mundo. Oracle Autonomous Database elimina la complejidad, el error humano y la gestión manual, y así contribuye a garantizar una mayor fiabilidad, seguridad y eficiencia al menor coste posible.

Ventajas de la base de datos autónoma
Las nubes de base de datos convencionales se gestionan y configuran de forma manual, son susceptibles de sufrir errores humanos y pueden quedar fácilmente expuestas a costosas vulnerabilidades de seguridad, haciendo que la promesa de ahorro y rentabilidad de la nube resulte imposible de cumplir. Oracle recurre al machine learning para revolucionar la gestión de datos con la introducción de la primera base de datos autónoma del mundo, que ofrece un mayor grado de automatización para incrementar el ahorro.

Productos Autonomous Database
Los productos de base de datos autónoma están disponibles en exclusiva a través de Oracle y se ofrecen en dos versiones optimizadas para satisfacer las necesidades específicas de procesamiento de transacciones online y almacenamiento de datos. Ambos utilizan machine learning y automatización para eliminar la complejidad, los errores humanos y la manipulación manual, por lo que garantizan una mayor fiabilidad, seguridad y eficiencia operativa. El servicio de procesamiento de transacciones autónomo permite un desarrollo y despliegue de aplicaciones mucho más sencillo para ofrecer capacidades de analítica en tiempo real, personalización y detección de fraudes. Autonomous Data Warehouse ofrece fiabilidad y rendimiento sin precedentes, así como una gestión de datos altamente flexible que hace posible el despliegue de almacenes de datos en cuestión de segundos.
